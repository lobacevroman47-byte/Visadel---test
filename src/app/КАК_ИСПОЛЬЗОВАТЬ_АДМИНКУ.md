# 🎯 Как использовать админ-панель Visadel Agency

## 🚀 Быстрый старт за 3 шага

### Шаг 1: Откройте админ-панель
Откройте файл `/admin.html` в браузере

### Шаг 2: Войдите с демо-доступом
```
Email: owner@visadel.agency
Пароль: owner123
```

### Шаг 3: Работайте!
Теперь у вас полный доступ ко всем разделам 🎉

---

## 🔑 Три уровня доступа

### 👑 Владелец (Owner)
**Доступ:** owner@visadel.agency / owner123
- ✅ Все разделы
- ✅ Добавление/удаление администраторов
- ✅ Настройки системы
- ✅ Изменение ролей

### 👨‍💼 Администратор (Admin)
**Доступ:** admin@visadel.agency / admin123
- ✅ Заявки
- ✅ Пользователи
- ✅ Страны и визы
- ✅ Конструктор анкет
- ❌ Нет доступа к настройкам

### 👤 Менеджер (Manager)
**Доступ:** manager@visadel.agency / manager123
- ✅ Только заявки
- ✅ Обработка виз
- ❌ Остальное недоступно

---

## 📋 Основные задачи

### 1️⃣ Обработать заявку клиента

**Для кого:** Все роли

1. Нажмите "Заявки" в левом меню
2. Найдите заявку (можно использовать поиск)
3. Нажмите на иконку 👁️ для просмотра
4. Измените статус на "В работе"
5. Когда виза готова:
   - Загрузите файл визы
   - Нажмите "Отправить пользователю"
6. Готово! Клиент получит уведомление

---

### 2️⃣ Начислить бонусы пользователю

**Для кого:** Owner, Admin

1. Откройте раздел "Пользователи"
2. Найдите нужного пользователя
3. Нажмите кнопку "Управление"
4. Введите сумму бонусов
5. Нажмите ➕ "Добавить"
6. Сохраните изменения

---

### 3️⃣ Добавить новую страну

**Для кого:** Owner, Admin

1. Перейдите в "Страны и визы"
2. Нажмите ➕ "Добавить страну"
3. Заполните форму:
   - Название страны
   - Флаг (emoji, например: 🇹🇭)
   - Тип визы
   - Цену
   - Срок действия
   - Время оформления
4. Включите галочку "Показывать в мини-апе"
5. Нажмите "Сохранить"

---

### 4️⃣ Создать анкету для страны

**Для кого:** Owner, Admin

1. Откройте "Конструктор анкет"
2. Выберите страну из списка
3. Нажмите ➕ "Добавить поле"
4. Настройте поле:
   - Название (например: "Фамилия")
   - Внутренний ключ (например: "lastName")
   - Тип поля (текст, число, дата и т.д.)
   - Поставьте галочку "Обязательное" если нужно
5. Нажмите "Сохранить поле"
6. Повторите для всех полей
7. Используйте drag-and-drop для изменения порядка
8. Нажмите "Сохранить изменения"

---

### 5️⃣ Добавить администратора

**Для кого:** Owner

1. Откройте раздел "Администраторы"
2. Нажмите ➕ "Добавить администратора"
3. Заполните форму:
   - Имя
   - Telegram @username
   - Выберите роль (Admin или Manager)
4. Нажмите "Добавить"

---

### 6️⃣ Настроить бонусы и цены

**Для кого:** Owner

1. Перейдите в "Настройки"
2. Измените нужные параметры:
   - Бонус новому пользователю
   - Бонус рефералу
   - Максимальное использование бонусов (%)
   - Цены на дополнительные услуги
3. Нажмите "Сохранить все изменения"

---

## 💡 Полезные советы

### 🔍 Поиск и фильтры
- Используйте поиск для быстрого нахождения заявок/пользователей
- Фильтры помогают отсортировать данные
- Можно комбинировать поиск и фильтры

### 🎨 Визуальные подсказки
- **Зелёные кнопки** = доступно
- **Серые кнопки** = нет прав доступа
- **Цветные badges** = статусы и роли
- При наведении появляются подсказки

### 💾 Сохранение
- В демо-версии все данные локальные
- В реальной системе будет автосохранение
- Сессия сохраняется между перезагрузками

### 🚪 Выход
- Кнопка "Выход" внизу бокового меню
- После выхода нужно войти заново

## ❓ Часто задаваемые вопросы

### Где посмотреть мои права доступа?
В боковом меню под логотипом показан ваш username и роль

### Почему некоторые разделы серые?
У вашей роли нет доступа к этим разделам. Войдите как Owner для полного доступа.

### Как изменить статус заявки?
Откройте заявку → выберите новый статус из выпадающего списка → сохраните

### Можно ли удалить владельца?
Нет, владельца удалить нельзя для безопасности

### Как пользователь получит визу?
После отправки визы пользователь автоматически получит уведомление в Telegram Mini App
