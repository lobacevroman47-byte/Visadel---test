# 🔐 Как попасть в админ-панель Visadel Agency

## ✅ САМЫЙ ПРОСТОЙ СПОСОБ

### Шаг 1: Откройте файл
```
admin-access.html
```

### Шаг 2: Нажмите кнопку
**"Войти в админ-панель"**

### Шаг 3: Войдите с демо-данными
```
Email: owner@visadel.agency
Пароль: owner123
```

**Готово!** Вы в админ-панели 🎉

---

## 🎯 Альтернативные способы

### Способ 2: Через URL параметр
Добавьте `?admin=true` к основному URL:
```
http://localhost:5173/?admin=true
```

### Способ 3: Через селектор
Откройте `index-selector.html` и выберите "Админ-панель"

## 🔑 Демо-учетные записи

### 👑 Владелец (Owner)
- **Email:** `owner@visadel.agency`
- **Пароль:** `owner123`
- **Доступ:** Все разделы без ограничений

### 👨‍💼 Администратор (Admin)
- **Email:** `admin@visadel.agency`
- **Пароль:** `admin123`
- **Доступ:** Все кроме настроек системы

### 👤 Менеджер (Manager)
- **Email:** `manager@visadel.agency`
- **Пароль:** `manager123`
- **Доступ:** Только раздел заявок

---

## 📋 Что делать после входа?

### Для первого знакомства (войдите как Owner):

1. **Dashboard** - посмотрите общую статистику
2. **Заявки** - откройте любую заявку и попробуйте изменить статус
3. **Пользователи** - попробуйте добавить/снять бонусы
4. **Страны и визы** - посмотрите список стран или добавьте новую
5. **Конструктор анкет** - создайте поля для анкеты
6. **Администраторы** - добавьте нового администратора
7. **Настройки** - настройте бонусы и цены

## ❓ Часто задаваемые вопросы

### Где находится админ-панель?
Откройте файл `admin-access.html` в корне проекта

### Какой доступ выбрать для теста?
Используйте **owner@visadel.agency** для полного доступа

### Можно ли создать свой аккаунт?
Сейчас только демо-аккаунты. В production будет регистрация через Owner

### Где хранятся данные?
Все данные сейчас mock (демо). В production будет база данных

### Как выйти из админки?
Кнопка "Выход" внизу левого меню
