# 🚀 Быстрый старт: Админ-панель Visadel Agency

## Как открыть админ-панель

### Вариант 1: Через селектор
Откройте `/index-selector.html` и выберите "Админ-панель"

### Вариант 2: Напрямую
Откройте `/admin.html` в браузере

---

## 🔑 Демо-учетные записи

Используйте одну из следующих учетных записей для входа:

### 👑 Владелец (полный доступ)
```
Email: owner@visadel.agency
Пароль: owner123
```

### 👨‍💼 Администратор
```
Email: admin@visadel.agency
Пароль: admin123
```

### 👤 Менеджер (только заявки)
```
Email: manager@visadel.agency
Пароль: manager123
```

## 📋 Что можно делать

### Как Owner (Владелец)
✅ Все разделы доступны
✅ Управление администраторами
✅ Изменение настроек системы
✅ Экспорт данных

### Как Admin (Администратор)
✅ Обработка заявок
✅ Управление пользователями
✅ Редактирование стран и виз
✅ Конструктор анкет
❌ Нет доступа к настройкам

### Как Manager (Менеджер)
✅ Только заявки
✅ Изменение статусов
✅ Загрузка виз
❌ Остальные разделы недоступны

---

## 🎯 Основные сценарии

### Обработать заявку
1. Войдите как Manager или выше
2. Откройте раздел "Заявки"
3. Нажмите на иконку 👁️ для просмотра
4. Измените статус на "В работе"
5. Загрузите готовую визу (PDF/IMG)
6. Нажмите "Отправить пользователю"

### Добавить страну
1. Войдите как Owner или Admin
2. Откройте "Страны и визы"
3. Нажмите "Добавить страну"
4. Заполните форму и сохраните

### Создать анкету
1. Войдите как Owner или Admin
2. Откройте "Конструктор анкет"
3. Выберите страну
4. Добавляйте поля анкеты
5. Настройте каждое поле
6. Сохраните изменения

### Управлять бонусами
1. Войдите как Owner или Admin
2. Откройте "Пользователи"
3. Нажмите "Управление" у нужного пользователя
4. Добавьте или снимите бонусы

### Добавить администратора
1. Войдите как Owner
2. Откройте "Администраторы"
3. Нажмите "Добавить администратора"
4. Заполните данные и выберите роль
