## 🎯 Особенности реализации

### Система ролей
```typescript
Owner > Admin > Manager
  3       2       1
```

- **Owner**: Полный доступ ко всем разделам
- **Admin**: Все кроме Настроек и управления ролями
- **Manager**: Только Заявки

### Защита маршрутов
Компонент `AdminLayout` автоматически проверяет права доступа и показывает:
- Экран "Доступ ограничен" для неавторизованных разделов
- Disabled кнопки в sidebar для недоступных разделов
- Подсказки "Недостаточно прав" при наведении

### Mock данные
Все данные работают на демо-данных из `/admin/data/mockData.ts`:
- 5 заявок с разными статусами
- 4 пользователя
- 8 стран
- 4 администратора

## 📱 Интеграция с Mini App

В реальной системе админ-панель будет синхронизироваться с Mini App через API:

### Уведомления
При отправке готовой визы пользователю:
```
Админ-панель → API → Telegram Bot → Пользователь
```

Текст уведомления:
> "Ваша виза готова. Вы можете скачать её в разделе Мои заявки"

### Синхронизация данных
- Заявки создаются в Mini App
- Обрабатываются в админ-панели
- Результат отображается в Mini App
