# 🎯 Функции админ-панели Visadel Agency

## 🔐 Авторизация

### Демо-аккаунты
```
Owner:   owner@visadel.agency   / owner123    (полный доступ)
Admin:   admin@visadel.agency   / admin123    (без настроек)
Manager: manager@visadel.agency / manager123  (только заявки)
```

---

## 📊 Dashboard

**Доступ:** Все роли

### Что показывает:
- 📈 Общее количество заявок
- 🔄 Заявки в работе
- 👥 Количество пользователей
- 🤝 Количество партнёров
- 💰 Общая выручка
- 📊 График распределения по статусам
- 📋 Последние 5 заявок

---

## 📝 Заявки

**Доступ:** Все роли

### Функции:
- 🔍 Поиск по имени, телефону, ID
- 🏷️ Фильтр по статусу (5 статусов)
- 🌍 Фильтр по стране
- 👁️ Просмотр полной информации
- 📄 Просмотр загруженных файлов
- ✏️ Изменение статуса
- 📤 Загрузка готовой визы
- 📧 Отправка визы пользователю

### Статусы:
1. **Черновик** - заявка не завершена
2. **Ожидает оплаты** - ждём оплату
3. **Ожидает подтверждения** - платёж на проверке
4. **В работе** - оформляем визу
5. **Готово** - виза отправлена

---

## 👥 Пользователи

**Доступ:** Owner, Admin

### Функции:
- 🔍 Поиск по всем полям
- 🏷️ Фильтр по статусу (Обычный/Партнёр)
- 💰 Управление бонусами:
  - ➕ Добавить бонусы
  - ➖ Снять бонусы
- 👤 Изменение статуса на Партнёра
- 🚫 Блокировка пользователя
- 📊 Просмотр истории заявок

### Данные пользователя:
- ФИО, Telegram, телефон, email
- Баланс бонусов
- Дата регистрации
- Количество заявок

---

## 🌍 Страны и визы

**Доступ:** Owner, Admin

### Функции:
- ➕ Добавление новой страны
- ✏️ Редактирование данных
- 🔄 Включение/отключение в Mini App
- 🔗 Переход к конструктору анкеты

### Параметры страны:
- Название и флаг (emoji)
- Тип визы
- Цена (₽)
- Срок действия
- Время оформления

## 📋 Конструктор анкет

**Доступ:** Owner, Admin

### Функции:
- 🌍 Выбор страны для редактирования
- ➕ Добавление полей
- ✏️ Редактирование полей
- 🗑️ Удаление полей
- 🔄 Drag-and-drop сортировка

### Типы полей:
1. **Текст** - обычный input
2. **Многострочный текст** - textarea
3. **Число** - number input
4. **Выпадающий список** - select с вариантами
5. **Дата** - date picker
6. **Диапазон дат** - две даты
7. **Чекбокс** - да/нет

### Настройки поля:
- Название (label)
- Внутренний ключ (служебное)
- Тип поля
- Обязательное/нет
- Placeholder
- Варианты (для select)

## 🚀 Быстрые действия

### Обработать заявку:
1. Заявки → 👁️ Просмотр
2. Статус → "В работе"
3. Загрузить визу
4. "Отправить пользователю"

### Добавить бонусы:
1. Пользователи → "Управление"
2. Ввести сумму
3. ➕ "Добавить"

### Создать страну:
1. Страны → ➕ "Добавить"
2. Заполнить форму
3. ✅ Включить показ
4. "Сохранить"

### Построить анкету:
1. Конструктор → Выбрать страну
2. ➕ Добавить поле
3. Настроить параметры
4. "Сохранить"
