# Админ-панель Visadel Agency

## 🔐 Доступ к админ-панели

Админ-панель доступна по отдельному URL: `/admin.html`

### Демо-учетные записи

#### Владелец (Owner)
- Email: `owner@visadel.agency`
- Пароль: `owner123`
- Права: Полный доступ ко всем разделам

#### Администратор (Admin)
- Email: `admin@visadel.agency`
- Пароль: `admin123`
- Права: Доступ к заявкам, пользователям, странам и анкетам

#### Менеджер (Manager)
- Email: `manager@visadel.agency`
- Пароль: `manager123`
- Права: Только заявки и их обработка

---

## 📋 Структура разделов

### 1. Dashboard (Главная)
- Статистика по заявкам
- Общая выручка
- Последние заявки
- Графики и аналитика

### 2. Заявки
- Таблица всех заявок с фильтрацией
- Просмотр деталей заявки
- Изменение статусов
- Загрузка готовых виз
- Отправка виз пользователям

**Статусы заявок:**
- Черновик
- Ожидает оплаты
- Ожидает подтверждения
- В работе
- Готово

### 3. Пользователи
**Доступ:** Owner, Admin

- Список всех пользователей
- Управление бонусами (добавить/снять)
- Изменение статуса (Обычный/Партнёр)
- Блокировка пользователей
- История заявок пользователя

### 4. Страны и визы
**Доступ:** Owner, Admin

- Список всех стран
- Добавление новых стран
- Редактирование данных о визах
- Управление ценами
- Включение/отключение в мини-апе

### 5. Конструктор анкет
**Доступ:** Owner, Admin

- Создание полей анкет для каждой страны
- Drag-and-drop сортировка полей
- Различные типы полей:
  - Текст
  - Число
  - Дата
  - Диапазон дат
  - Выпадающий список
  - Чекбокс
  - Многострочный текст

### 6. Администраторы и роли
**Доступ:** Owner, Admin

- Список всех администраторов
- Добавление новых администраторов
- Изменение ролей
- Блокировка/разблокировка
- Удаление администраторов

### 7. Настройки
**Доступ:** Owner

- Реферальная система (бонусы)
- Бонусная система (лимиты)
- Цены на дополнительные услуги
- Экспорт заявок в Excel

- Модальные окна для редактирования

## 🔒 Система ролей

### Owner (Владелец)
- Полный доступ ко всем разделам
- Управление администраторами
- Изменение ролей
- Доступ к настройкам системы

### Admin (Администратор)
- Заявки, пользователи
- Страны и анкеты
- Бонусная система
- Нет доступа к настройкам и управлению ролями

### Manager (Менеджер)
- Только раздел "Заявки"
- Изменение статусов
- Загрузка и отправка виз
- Нет доступа к остальным разделам

## 🚀 Основные функции

### Обработка заявки (для менеджеров)
1. Открыть раздел "Заявки"
2. Найти нужную заявку
3. Нажать на иконку "Просмотр"
4. Изменить статус на "В работе"
5. Загрузить готовую визу
6. Нажать "Отправить пользователю"
7. Статус автоматически меняется на "Готово"

### Управление бонусами пользователя
1. Открыть раздел "Пользователи"
2. Найти нужного пользователя
3. Нажать "Управление"
4. Ввести сумму бонусов
5. Нажать "Добавить" или "Снять"

### Добавление новой страны
1. Открыть раздел "Страны и визы"
2. Нажать "Добавить страну"
3. Заполнить все поля
4. Включить тумблер "Показывать в мини-апе"
5. Сохранить

### Создание анкеты для страны
1. Открыть раздел "Конструктор анкет"
2. Выбрать страну
3. Нажать "Добавить поле"
4. Настроить параметры поля
5. Сохранить поле
6. Упорядочить поля drag-and-drop
7. Сохранить изменения
